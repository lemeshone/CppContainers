# CppContainers

Реализация стандартных контейнеров STL на C++20 в учебных целях.  
В проекте реализованы последовательные, ассоциативные и адаптеры контейнеров с интерфейсом, максимально близким к стандартной библиотеке.

---

## Реализованные контейнеры

Папка `src/Containers`:

- **Array/**
- **Vector/**
- **List/**
- **Queue/**
- **Stack/**
- **Associative/**
  - `s21_set.h`
  - `s21_multiset.h`
  - `s21_map.h`
  - `s21_rbt.h` — внутреннее красно-чёрное дерево

Общие заголовки:

- `s21_containers.h` 
- `s21_containersplus.h` 

Все контейнеры написаны на шаблонах и стараются повторять поведение стандартных (`std::vector`, `std::list`, `std::map`, `std::set` и т.д.).

---

## Технологии

- **C++20**
- Компилятор: `g++`
- Тесты: **Google Test** (`gtest`, `gtest_main`)
- Отладка: `-ggdb`, `_GLIBCXX_DEBUG`
- Утилиты: `make`, `valgrind` (опционально)
